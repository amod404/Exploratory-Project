# Lamarckian Evolutionary algorithm for Multi-Objective Neural Architecture DEsign. (LEMONADE)

An efficient, evolutionary approach to Neural Architecture Search utilizing Lamarckian inheritance to optimize the Pareto front of model architectures.

## 📖 About The Project
This repository contains the implementation of a Lamarckian-based Neural Architecture Search framework. Unlike standard Darwinian approaches, this method allows child networks to inherit not just the architectural traits (genotype) but also the learned weights (phenotype) from their parents. This dramatically reduces the training overhead and accelerates the discovery of optimal architectures balancing accuracy, parameters, and FLOPs.

## 🚀 Getting Started

Follow these instructions to set up your local development environment and run the search algorithms.

### Prerequisites

Ensure you have Python installed on your system and have cloned the repo. This project uses virtual environments to manage dependencies cleanly.

### Installation & Setup

1. **Navigate to the desired version directory:**
   Choose the version of the NAS you want to run (e.g., `version1`, `version2`) and change into that directory.
   
   ```bash
   cd version.x
   ```
   *(I prefer to open specific folder in your editor)*

2. **Create a virtual environment:**
   
   ```bash
   python -m venv myVenv
   ```

3. **Activate the virtual environment:**
   * **Windows:**
     ```bash
     .\myVenv\Scripts\activate
     ```
   * **Linux/macOS:**
     ```bash
     source myVenv/bin/activate
     ```

4. **Install the required dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

## ⚙️ Configuration & Usage

Before initiating the architecture search, you can tune the hyperparameters (such as epochs, learning rates, batch size, and the ENAS controller settings). 

1. Open `config.py` in your text editor.
2. Adjust the hyperparameters and settings to fit your hardware constraints and target dataset.
3. Save the file.

### To start the NAS pipeline, run:

```bash
python main.py
```

## 📊 Interactive Visualization

To analyze the performance, parameter efficiency, and the Pareto front of the generated models, we provide an interactive plotting utility.

1. Locate the timestamped result folder generated by your run (e.g., `results_CIFAR-100_20260419_093906`).
2. Open `util/plot22.py`.
3. Update the folder name variable inside the script to match your specific result timestamp.
4. Run the visualization script from your terminal:
   
   ```bash
   python .\util\plot22.py
   ```
This will launch an interactive dashboard where you can explore the 3-D visiualization of each Generation.
